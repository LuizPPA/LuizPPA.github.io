Coloração de Grafos

Juntamente com um colega de trabalho, concluí no último bimestre de 2015 um projeto envolvendo coloração de Grafos. O projeto foi escrito em C e entregue para a conclusão da matéria AEDS (Algorítimos e Estruturas de Dados). O problema proposto pelo projeto era organizar uma turma de alunos para que aqueles que tem afinidade entre si, não se sentassem próximos uns dos outros durante a prova. Quando é inevitável ter alunos com afinidade próximos uns dos outros, o programa sugere a criação de uma nova prova, como no caso crítico, onde todos os alunos têm afinidade. O trabalho teve como objetivo aprimorar o uso da etrutura de dados de grafos e comprender o conceito de complexidade de algorítimos fazendo o estudo de complexidade. Ao lado um link para o download dos códigos.

Machina Speculatrix

Machina Speculatrix foi o nome dado ao robô autômato que está sendo atualmente desenvolvido pelo grupo de robótica MS - Robótica do Coltec UFMG do qual eu juntamente com dois colégas coordeno. O autômato Machina Speculatrix é um dos três robos em fase de construção no grupo com objetivo de competir pela Competição de Robôs Automatos. A idéia de se fazer três robôs surgiu de uma discução entre os integrantes do grupo acerca de qual tipo de robô seria eficiente: Um tricíclo, um quadricíclo com eixo diferêncial ou um robô com duas rodas e uma pololu (que é o robô no qual trabalho). Ao lado, um link para download de uma melhor descrição do projeto.

Portal de Vídeos

Um dos meus projetos mais abrangentes e desafiadores (não é atoa que foi me passado como um desafio). O projeto consistia em finalizar as funcionalidades da simulação um portal de vídeos para o colégio ende curso informática. Foi entregue um sistema (com frontend pronto porém customizável e backend interminado) a mim e outros colégas para que individualmente completassemos as funcionálidades que faltavam. A idéia era fazer um portal de vídeos para o Coltec onde os usuários fossem capazes de adicionar, comentar, avaliar e assistir vídeos. Os códigos foram escritos html, javascript e php (portanto será necessário um servidor para abrir o projeto). Em seguida os códigos para download.
